# Launch Curtain 0.2.0

First public release.

Launch Curtain is a small Decky Loader plugin for Windows. It shows a simple
console-style loading screen while Steam Big Picture starts a game, hiding the
desktop and launcher noise that normally appears during PC game launches.

## Highlights

- Clean black loading curtain with centered logo and animated dots.
- Immediate in-Steam transition when a game starts.
- Fullscreen Windows overlay for launcher and desktop flashes.
- Automatic Decky panel translations.
- Custom logo picker.
- Does not enable or change Steam Overlay settings.

## Supported Languages

English, Italian, French, Spanish, Portuguese, Brazilian Portuguese, German,
Dutch, Ukrainian, Chinese, and Japanese.

## Install

Download `LaunchCurtain-0.2.0-installabile.zip`, extract it, and copy the
`LaunchCurtain` folder to your Decky Loader plugins folder.
